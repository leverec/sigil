# Code of Conduct

## Our Standards

This project is a small, open tool built and maintained by individuals who give their time voluntarily. To keep the environment productive and welcoming for everyone, the following standards apply to all interactions in this project, including issues, pull requests, discussions, and commit messages.

Behavior that is expected and encouraged:

- Being respectful and considerate in all communication
- Giving and receiving constructive feedback gracefully
- Focusing criticism on code and ideas, not on people
- Acknowledging mistakes and moving forward without hostility
- Helping others understand the project when they ask in good faith

Behavior that is not acceptable:

- Personal attacks, insults, or demeaning language toward any contributor or user
- Harassment of any kind, public or private
- Dismissing or belittling contributions without explanation
- Deliberately derailing discussions or pull request reviews
- Publishing private information about someone without their consent

## Scope

This Code of Conduct applies within all project spaces: the GitHub repository, issues, pull requests, and any other official communication channel associated with Sigil.

## Enforcement

Instances of unacceptable behavior may be reported by opening a private message or contacting the maintainer directly through GitHub. All reports will be reviewed and handled at the maintainer's discretion.

Consequences for violations may include:

- A written warning
- Temporary restriction from contributing
- Permanent removal from the project

The maintainer reserves the right to remove, edit, or reject any contribution, comment, or issue that does not align with this Code of Conduct.
